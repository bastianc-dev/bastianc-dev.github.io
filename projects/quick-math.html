Quick Math

Aplicación responsive para practicar cálculo mental con distintas operaciones, niveles de dificultad, modos infinito y contrarreloj, y récords guardados en el navegador.

01 · Contexto

Descripción

Quick Math permite practicar cálculo mental mediante ejercicios generados dinámicamente. Antes de comenzar, el usuario selecciona la operación, el rango numérico y el modo de juego.

La métrica principal es el promedio de segundos por operación correcta. Las respuestas equivocadas no cambian de ejercicio y la última operación incompleta no se contabiliza.

02 · Producto

Funcionalidades

  • Ejercicios de suma, resta, multiplicación y división.
  • Selección de rangos numéricos.
  • Modo infinito.
  • Modo contrarreloj de uno, tres o cinco minutos.
  • Avance automático al responder correctamente.
  • Cálculo del promedio de segundos por operación.
  • Récords guardados por configuración.
  • Interfaz adaptada a dispositivos móviles.

04 · Criterio técnico

Decisiones técnicas

JavaScript sin framework

El proyecto se construyó con DOM, eventos y estado propio para practicar los fundamentos del lenguaje.

localStorage

Los récords permanecen en el navegador sin necesitar una base de datos o un backend.

Divisiones exactas

El divisor y el cociente se generan primero para asegurar que el resultado sea entero.

Promedio por aciertos

El resultado se calcula solo con operaciones correctas para representar mejor el rendimiento.

05 · Resultado

Resultado
